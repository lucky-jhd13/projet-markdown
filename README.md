
# ⚽ Appli Prono (Pro-Foot AI)

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white) ![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)

## 📝 Description
Application avancée d'analyse et de pronostics pour le football. Basée sur le modèle de Dixon-Coles et différents algorithmes d'ELO, l'application estime la force des équipes pour calculer des prévisions de score viables.

## 🚀 Fonctionnalités
- Interfaces dynamiques et responsives avec **Streamlit**.
- Algorithmes pointus de prédiction.
- Suggestions de paris à forte valeur (Value Bets).

## 🛠️ Requirements & Lancement
```bash
pip install -r requirements.txt
streamlit run app.py
```
